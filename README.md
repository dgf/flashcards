## Create a flashcard PDF from a dictionary

```sh
./flashcards -h
Usage of ./flashcards:
  -config string
    	configuration file (default "config.toml")
  -dict string
    	card dictionary file (default "dicts/es-de.dict")
  -path string
    	output PDF file (default "out.pdf")

```
